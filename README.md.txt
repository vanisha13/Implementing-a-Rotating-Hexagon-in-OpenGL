# SimpleOpenGLProject

This is a simple OpenGL project using GLEW and GLFW.

## Building

1. Ensure you have CMake installed.
2. Create a build directory and navigate to it:
    ```sh
    mkdir build
    cd build
    ```
3. Run CMake to configure the project:
    ```sh
    cmake ..
    ```
4. Build the project:
    ```sh
    cmake --build .
    ```

## Running

Run the executable `SimpleOpenGLProject.exe` generated in the `bin` directory.

## Dependencies

- GLEW
- GLFW
